# Prompt Library — saved master prompts for recurring jobs

Ready-to-paste, optimized prompts for the jobs that come up again and again, so the
best prompt is **reused, not re-derived**. Each was built with the `prompt-architect`
skill (its playbook applied *selectively* — only the techniques the job needs).

**How to use:** open the file, copy the prompt block, fill the `{placeholders}`, paste
it where the AI runs (a chat, an API call like `IDKWhatimdoing`'s, a CMS field).

**How to grow it:** when `prompt-architect` crafts a new prompt that works well for a
recurring job, save it here in the same format (job → prompt → techniques → acceptance
test). A prompt earns a place only after it produced a good result on a real job.

| Prompt | Use it for |
|---|---|
| [`biho-edit-assistant.md`](biho-edit-assistant.md) | The CMS "improve this text" button — polish ONE field of a client's live site, drop-in and truthful. |
| [`local-service-copy.md`](local-service-copy.md) | Write a fresh section of copy (hero, service blurb, about, CTA) for a local-service business from a short brief. |
| [`seo-metadata.md`](seo-metadata.md) | A page's `<title>` + meta description — char-limited, local-SEO, click-worthy. |
| [`lead-reply.md`](lead-reply.md) | Draft a fast, warm reply to a customer estimate/inquiry that gathers what's needed to quote. |

Every prompt here shares two non-negotiables for client-facing work: **use only the
facts given (never invent services, prices, reviews, or awards)** and **stay in the
owner's plain, no-hype voice**.
